== Description ==

Disable Autosave prevents WordPress from automatically saving duplicate copies of posts while editing.

== Installation ==

BEFORE YOU BEGIN: Requires the installation and activation of [LAPDI Easy Dev Latest Version](http://wordpress.org/plugins/tsp-easy-dev)

1. Upload `tsp-disable-auto-save` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= I've installed the script but nothing happens. What did I do wrong? =

Nothing. The script runs in real-time behind the scenes. Once installed and activated, there are no other steps to take.
